What's this about?

This website uses elections.ca to make it a little easier for Canadians to find out where they should vote on voting day, and displays that information in a way that I hope is also more readable.

Because this site piggybacks off the government site, the information displayed here should be accurate.

To find out more about what's going on behind the scenes, the entire project can be viewed on Github.

If you find any bugs, you can open an issue on the Github repo or email me.
